Installing http-server

Step 1: Install Node.js Before we can install and use http-server, we need to install Node.js. Node.js is a JavaScript runtime that allows us to run JavaScript programs on our computers. Follow these steps to install Node.js:

  1. Open your web browser and go to the official Node.js website: https://nodejs.org.
  1. On the homepage, you'll see two versions to choose from: LTS (Long-Term Support) and Current. For stability, it's recommended to download the LTS version.
  1. Click on the "Downloads" button for the LTS version.
  1. On the next page, you'll see different installers for various operating systems (Windows, macOS, Linux). Choose the installer suitable for your computer's operating system.
  1. Once the installer is downloaded, run it and follow the installation instructions. The default settings should work fine for most cases.

Step 2: Open Visual Studio Code Make sure you have Visual Studio Code (VS Code) installed on your computer. If you don't have it, you can download it from the official website: https://code.visualstudio.com.

Step 3: Open the Terminal in VS Code

  1. Open Visual Studio Code.
  1. In the top menu, click on "View" and select "Terminal" from the drop-down menu. Alternatively, you can press the shortcut keys Ctrl + on Windows/Linux or Cmd + on macOS.

Step 4: Install http-server In the Terminal, follow these steps to install http-server:

  1. Type the following command in the Terminal and press Enter:
npm install -g http-server

This command tells Node.js to install the http-server package globally on your computer. The -g flag means it will be available system-wide.

  1. Wait for the installation to complete. You'll see some progress information and messages in the Terminal.

Step 5: Start a Simple Web Server Now that http-server is installed, let's start a simple web server to serve your HTML, CSS, and JavaScript files:

  1. Navigate to the folder where your website files are located. You can use the cd command followed by the folder path. For example, if your files are on the desktop, type:
cd Desktop. (Read above, Desktop is just a placeholder for where your files are)
  1. Once you're in the correct folder, type the following command and press Enter:
http-server

This command starts the http-server and tells it to serve the files in the current folder.

  1. You'll see some output in the Terminal indicating that the server is running. It will display the local IP address and the port number where your server is accessible (usually http://localhost:8080).

Step 6: Open Your Website in the Browser To see your website in action, follow these steps:

  1. Open your preferred web browser (Chrome, Firefox, etc.).
  1. In the address bar, type http://localhost:8080 (or the appropriate address and port shown in the Terminal) and press Enter.
  1. Your website should now be displayed in the browser. You can interact with it just like any other website.

Congratulations! You've successfully installed and used http-server in Visual Studio Code to host your website locally. Remember, the server will keep running as long as the Terminal window is open. If you want to stop the server, you can close the Terminal
