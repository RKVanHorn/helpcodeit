Key Terms for HTML

  • Element: The basic building blocks of HTML.
  • Tag: Markup code that defines elements and attributes.
  • Attribute: Additional information about an element, defined within the tag.
  • HTML Document Structure: The arrangement of <!DOCTYPE>, <html>, <head>, and <body> tags.
  • Meta Tags: Information about the HTML document, placed within <head>.
  • Hyperlink: An element that links to another resource, typically using the <a> tag.
  • Text Formatting: Elements like <b>, <i>, and <strong> to format text.
  • List: Ordered (<ol>) or unordered (<ul>) lists.
  • Table: Represents tabular data, made up of <table>, <tr>, <td>, and optionally <th> tags.
  • Form: Elements such as <form>, <input>, and <textarea> for user input.
  • CSS (Cascading Style Sheets): Used for styling HTML elements.
  • ID: Unique identifier for an element.
  • Class: used to apply a style to an individual item but made in a way that is reusable.
  • Inline vs Block Elements: Elements that take up the full width (block) or only the necessary width (inline).
  • Semantic Tags: Elements like <header>, <footer>, <article>, and <section> that carry meaning.
  • Media Elements: <img>, <audio>, <video> for embedding media.
  • Accessibility: ARIA roles, alt attributes, and other practices to make web pages more accessible.
  • DOCTYPE: Declares the document type and version of HTML.
  • Viewport: Meta tag for responsive design.
  • HTML Entities: Special characters represented in HTML, like &amp; for &.
  • Comments: Code ignored by browsers, used for notes (<!-- Comment -->).
