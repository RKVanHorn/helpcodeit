Array Methods

Method Name Return Value Explanation Example
push() The modified array Appends one or more elements to the end of the array. arr.push("hello", "world");
pop() The removed element Removes the last element from the array and returns it. const lastElement = arr.pop();
shift() The removed element Removes the first element from the array and returns it. const firstElement = arr.shift();
unshift() The modified array Prepends one or more elements to the beginning of the array. arr.unshift("hello", "world");
splice() A new array with the removed elements Removes elements from the array and optionally inserts new elements at the specified indexes. const newArr = arr.splice(1, 2, "hello", "world");
slice() A new array with the specified elements Copies a section of the array and returns it. const newArr = arr.slice(1, 3);
concat() A new array with the concatenated arrays Concatenates two or more arrays and returns a new array. const newArr = arr.concat(["hello", "world"]);
join() A string with the elements of the array Joins the elements of the array into a string. const string = arr.join(", ");
map() A new array with the results of the function Applies a function to each element of the array and returns a new array with the results. const newArr = arr.map(element => element * 2);
filter() A new array with the elements that pass the test Creates a new array with the elements of the original array that pass a test. const newArr = arr.filter(element => element % 2 === 0);
reduce() The accumulated value Reduces the array to a single value by applying a function to each element and accumulating the results. const sum = arr.reduce((accumulator, element) => accumulator + element, 0);
reduceRight() The accumulated value Reduces the array to a single value by applying a function to each element in reverse order and accumulating the results. const sum = arr.reduceRight((accumulator, element) => accumulator + element, 0);
every() True or false Tests whether all elements in the array pass the test implemented by the provided function. const allEven = arr.every(element => element % 2 === 0);
some() True or false Tests whether some element in the array passes the test implemented by the provided function. const someEven = arr.some(element => element % 2 === 0);
includes() True or false Tests whether the array contains the specified element. const hasElement = arr.includes(2);
indexOf() The index of the element Returns the first index of the specified element in the array. const index = arr.indexOf(2);
lastIndexOf() The index of the element Returns the last index of the specified element in the array. const index = arr.lastIndexOf(2);
find() The value of the element Returns the value of the first element in the array that satisfies the provided testing function. const element = arr.find(element => element % 2 === 0);
findIndex() The index of the element Returns the index of the first element in the array that satisfies the provided testing function. const index = arr.findIndex(element => element % 2 === 0);
sort() The modified array Sorts the elements of the array in place. arr.sort();
reverse() The modified array Reverses the order of the elements of the array in place. arr.reverse();
fill() The modified array Fills all the elements of the array from a start index to an end index with a static value. arr.fill(0, 1, 3);
copyWithin() The modified array Copies a sequence of array elements within the array. arr.copyWithin(0, 1, 3);
flat() A new array with the sub-array elements concatenated Creates a new array with all sub-array elements concatenated into it recursively up to the specified depth. const newArr = arr.flat();
flatMap() A new array with the sub-array elements concatenated Maps each element using a mapping function, then flattens the result into a new array. const newArr = arr.flatMap(element => [element * 2]);
