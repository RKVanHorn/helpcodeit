Key Terms for JavaScript

  • Variable: A container for a value, defined with let or const.
  • String: A sequence of characters, defined with single or double quotes.
  • Number: A numeric value, defined with or without decimals.
  • Boolean: A value that is either true or false.
  • Array: A list of values, defined with square brackets.
  • Object: A collection of key-value pairs, defined with curly braces.
  • Function: A reusable block of code, defined with the function keyword.
  • Method: A function that is a property of an object.
  • Operator: A symbol that performs an operation on one or more values.
  • Comparison Operator: An operator that compares two values and returns a boolean.
  • Conditional Statement: A statement that runs different code depending on a condition.
  • Logical Operator: An operator that combines multiple boolean values. Examples include &&, ||, and !.
  • Loop: A statement that runs code repeatedly.
  • Event: An action that occurs in the browser, such as a click or keypress.
  • Event Listener: A function that runs when a specific event occurs.
  • DOM (Document Object Model): A representation of the HTML document, which can be modified with JavaScript.
  • DOM Node: An element, attribute, or text in the DOM.
  • DOM Query: A method that returns a DOM node.
  • DOM Manipulation: Changing the DOM with JavaScript.
  • DOM Event: An event that occurs on a DOM node.
  • DOM Event Listener: A function that runs when a DOM event occurs.
  • DOM Event Handler: A function that runs when a DOM event occurs.
  • DOM Event Propagation: The order in which event handlers are triggered.
  • DOM Event Bubbling: Event handlers on child elements are triggered before event handlers on parent elements.
  • DOM Event Capturing: Event handlers on parent elements are triggered before event handlers on child elements.
  • DOM Event Delegation: Attaching a single event listener to a parent element to handle events on child elements.
  • DOM Event Object: An object that contains information about a DOM event.
  • DOM Event Prevent Default: A method that prevents the default behavior of an event.
