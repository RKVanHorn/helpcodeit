Concatenation in JavaScript

Concatenation is a fancy word for joining things together, and in the world of JavaScript, it means connecting strings of text. Imagine you have different words or sentences and you want to link them to make a complete thought, just like building a train by connecting carriages. That's what concatenation does! We use it to create personalized messages, assemble sentences from separate pieces, or even add numbers to text. It's like gluing words together to make them say exactly what we want. Whether you're building a greeting for a website visitor or putting together a full paragraph from smaller parts, concatenation is a handy tool that helps you write smarter and more flexible code.

There are two ways to concatenate strings in JavaScript: using the plus sign (+) or using template literals. We'll cover both in this lesson.

String Concatenation

Below is an example of concatenation. The first line creates a variable called string1 and assigns it the value of "Hello, ". The second line creates a variable called string2 and assigns it the value of "World!". The third line creates a variable called combinedString and assigns it the value of string1 plus string2. The result is a new string that says "Hello, World!".


let string1 = "Hello, ";
let string2 = "World!";
let combinedString = string1 + string2; // "Hello, World!"

Adding a Number to a String

Below is an example of concatenation with a number. The first line creates a variable called string and assigns it the value of "I have ". The second line creates a variable called number and assigns it the value of 5. The third line creates a variable called completeSentence and assigns it the value of string plus number plus " apples.". The result is a new string that says "I have 5 apples.".


let string = "I have ";
let number = 5;
let completeSentence = string + number + " apples."; // "I have 5 apples."

Without Spaces

Below is an example of concatenation without spaces. We create a variable called noSpaces and assign it the value of "Hello," + "World!" with no spaces left before or after each. Make sure when using concatenation you take this into account to prevent this issue.


let noSpaces = "Hello," + "World!"; // "Hello,World!"

Template Literals

Template literals are a new way to write strings in JavaScript. They allow you to write multi-line strings and include variables without having to use concatenation. Template literals are enclosed in backticks (``) instead of quotes ("" or ''). When you want to include a variable, you use the dollar sign and curly braces (${}) to wrap the variable name. This is called string interpolation.

      
let name = "Alice";
let age = 12;
let introduction = `My name is ${name}, and I am ${age} years old.`; // "My name is Alice, and I am 12 years old."

With template literals, you can write more complex sentences easily!
