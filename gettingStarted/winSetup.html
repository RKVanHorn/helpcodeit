Windows Setup

1. Install Node.js and npm

Go to the official Node.js website and download the installer suitable for your system. By installing Node.js, you'll automatically get npm as well. 

node -v
npm -v

Run the above commands in your terminal to verify the installation. If installed correctly, you should see the version of Node.js and npm installed on your system.

1. Install Git

Download the Git installer for Windows from the official Git website. Follow the installation instructions. 

git --version

Use the above command to check if Git was installed successfully. If installed correctly, you should see the version of Git installed on your system.
